Nginx access_log 条件过滤

2020-10-26 1 min read # Linux

按 IP 正则过滤

//tips： nginx版本 >= 1.7.0

在http{} 加入
           
map $http_x_forwarded_for $iploggable{
    // 1：true  0：false
   default 1;
   ~^47.107.15.238 0;
}

access_log  /var/log/nginx_access.log  main if=$iploggable;

按 Request 过滤

在server{} 加入

location ~* ^.+\.(gif|jpg|png|css|js|map|woff)$ {
    access_log off;
}
      • 按 IP 正则过滤
      • 按 Request 过滤
